# Screenshots Guide for Play Store

This guide explains how to capture and prepare screenshots for the Google Play Store listing.

## üìã Requirements

### Phone Screenshots (Required)
- **Minimum**: 2 screenshots
- **Maximum**: 8 screenshots
- **Size**: 1080 x 1920 pixels (9:16 ratio) or 1080 x 2340 for taller screens
- **Format**: PNG or JPEG (24-bit, no alpha)
- **File size**: 8MB max per screenshot

### 7-inch Tablet Screenshots (Optional)
- **Size**: 1200 x 1920 pixels
- **Format**: PNG or JPEG

### 10-inch Tablet Screenshots (Optional)
- **Size**: 1600 x 2560 pixels
- **Format**: PNG or JPEG

## üé¨ Recommended Screenshots

Create screenshots for these 8 key screens in order:

### 1. **Home Screen** (Priority: High)
**What to show**:
- Top app bar with search icon and menu (three dots)
- Category filter chip (shows current category)
- Three content sections with horizontal carousels:
  - Channels section with "See All" link
  - Playlists section with "See All" link

4. Take screenshot (Power + Volume Down)

### 2. **Video Player** (Priority: High)
**What to show**:
- Video playing in portrait mode

  - Title, author name, view count

**How to capture**:
1. Tap any video from Home to open player

**Caption**: "Advanced player with quality selection, Chromecast, PiP, and gesture controls"

## üì± Important App Details for Screenshots

### Bottom Navigation (5 Tabs)
The app has **5 tabs** in the bottom navigation bar (from left to right):
1. **Home** - Shows carousels of channels, playlists, and videos
2. **Channels** - List of all channels with FAB for category filtering
3. **Playlists** - List of all playlists
4. **Videos** - Grid view of all videos (2 columns on phone, more on tablets)
5. **Downloads** - Downloaded videos and library sections

### Onboarding Flow
When you first launch the app, you'll see:
1. Splash screen (250ms)
2. Onboarding carousel (3 pages) - Skip this to get to main app
3. Main app with Home screen

**Tip**: To skip onboarding after first install, you may need to clear app data or use a device that already has the app installed.

### Video Player Features
The player screen includes many advanced features:
- **Quality Selection**: Tap quality button to see available resolutions (360p, 720p, 1080p, etc.)
- **Chromecast**: Cast button appears if Chromecast device is available
- **Subtitles**: Subtitle button to select captions/subtitles
- **Fullscreen**: Toggle fullscreen mode
- **Up Next Queue**: Shows recommended videos below player
- **Gesture Controls**: Brightness (left swipe), volume (right swipe), seek (horizontal swipe)

### Backend Configuration
Make sure the backend is running before capturing screenshots:
```bash
# In backend folder
./gradlew bootRun

# Backend should be accessible at:
# - For physical device: http://YOUR_IP:8080
# - For emulator: http://10.0.2.2:8080
```

Check `android/app/build.gradle.kts` for the correct `API_BASE_URL` configuration.

### Sample Data
The app requires content to be seeded in the backend. Run the seed script:
```bash
cd backend
./gradlew bootRun --args='--spring.profiles.active=seed'
```

This will populate:
- 19 categories
- 13 channels
- 6 playlists
- 173 videos

### Downloads Feature
To capture download screenshots:
1. Open any video in the player
2. Scroll down to see action buttons
3. Tap the Download button
4. Accept EULA if prompted
5. Video will start downloading
6. Navigate to Downloads tab to see progress
7. Completed downloads show "Open" button

## üõ†Ô∏è Capture Methods

### Method 1: Physical Device (Recommended)
```bash
# 1. Connect device via USB with ADB enabled
# 2. Install debug APK:
cd android
./gradlew installDebug

# 3. Open app and navigate to each screen
# 4. Take screenshots using device (Power + Volume Down)
# 5. Pull screenshots from device:
adb pull /sdcard/Pictures/Screenshots/ ./play-store/screenshots/
```

### Method 2: Android Emulator
```bash
# 1. Create Pixel 6 emulator (1080x2340) in Android Studio
# 2. Launch emulator
# 3. Install APK:
adb install app/build/outputs/apk/debug/app-debug.apk

# 4. Navigate and use emulator screenshot button (camera icon)
# 5. Screenshots saved to: ~/Library/Android/sdk/screenshots/
```

### Method 3: Android Studio Layout Editor
- Open layout XML files
- Use "Preview" tab
- Click camera icon to capture
- Less realistic, not recommended for Play Store

## ‚úÖ Pre-Upload Checklist

Before uploading to Play Store:

**Required (minimum 2, recommended 8):**
- [ ] 8 main phone screenshots captured (Home, Player, Videos, Search, Channel Detail, Channels, Downloads, Settings)
- [ ] Screenshots are 1080 x 1920 (or device native resolution like 1080 x 2340)
- [ ] Screenshots show actual app content (not mockups)
- [ ] Backend is running and app shows real data (not empty states)
- [ ] No personal information visible
- [ ] Content is appropriate and accurate
- [ ] Screenshots are high quality (no blur, no artifacts)
- [ ] File sizes under 8MB each
- [ ] Files named descriptively (01-home.png, 02-player.png, etc.)
- [ ] Screenshots ordered logically (most important first)

**Optional but Recommended:**
- [ ] Captions prepared for each screenshot (see suggested captions above)
- [ ] Localized screenshots for Arabic (ar) showing RTL layout
- [ ] Localized screenshots for Dutch (nl)
- [ ] Bonus screenshots (Playlist Detail, Categories, Fullscreen Player, Onboarding)
- [ ] Tablet screenshots if targeting tablets (7-inch and/or 10-inch)

**Technical Checks:**
- [ ] App connects to backend successfully (check API_BASE_URL in build.gradle.kts)
- [ ] Backend has seeded data (channels, playlists, videos visible in app)
- [ ] Video player can play videos (stream extraction works)
- [ ] Downloads feature works (can download at least one video)
- [ ] Search returns results
- [ ] All bottom navigation tabs show content

## üöÄ Quick Start

Fastest way to get screenshots:

```bash
# 1. Install app on device
cd android
./gradlew installDebug

# 2. Open app and navigate to 8 screens listed above:
#    - Home (default screen after onboarding)
#    - Player (tap any video)
#    - Videos tab (4th bottom nav tab)
#    - Search (tap search icon on Home)
#    - Channel Detail (tap any channel)
#    - Channels tab (2nd bottom nav tab)
#    - Downloads (5th bottom nav tab)
#    - Settings (menu on Home ‚Üí Settings)

# 3. Take screenshots (Power + Volume Down on most devices)

# 4. Pull screenshots from device
adb pull /sdcard/DCIM/Screenshots/ ./play-store/screenshots/phone/en/
# Or:
adb pull /sdcard/Pictures/Screenshots/ ./play-store/screenshots/phone/en/

# 5. Rename files in chronological order:
mv Screenshot_20250101_120001.png 01-home.png
mv Screenshot_20250101_120002.png 02-player.png
mv Screenshot_20250101_120003.png 03-videos-tab.png
mv Screenshot_20250101_120004.png 04-search.png
mv Screenshot_20250101_120005.png 05-channel-detail.png
mv Screenshot_20250101_120006.png 06-channels-tab.png
mv Screenshot_20250101_120007.png 07-downloads.png
mv Screenshot_20250101_120008.png 08-settings.png

# 6. Upload to Play Console
```

Done! You now have 8 screenshots ready for Play Store submission.
